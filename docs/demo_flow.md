# Flujo de Demo

1. Ejecutar `python src/ingestion/load_data.py`.
2. Iniciar dashboard con `python src/app/main.py`.
3. Mostrar KPIs y bandeja.
4. Abrir un caso rojo crítico y mostrar score + reglas + evidencia.
5. Ejecutar preguntas del jurado en el copiloto:
   - top 10 riesgo
   - por qué este siniestro fue marcado
   - proveedores que concentran alertas
   - qué revisar primero y por qué
6. Mostrar que el copiloto cita reglas y playbooks del corpus interno.
7. Mostrar el simulador con un caso dirigido.
8. Cerrar con la nota ética: alerta de revisión, no acusación.
